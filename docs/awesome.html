Список докладов, статей и пакетов, рекомендованных к ознакомлению.

Доклады​

  • Назим Гафаров — Воркшоп: Знакомство с MobX
  • MobX, or: Going from Mutable to Immutable to Reactive State Management - От автора Mobx. Сначала показывается наивный мутабельный подход управления состоянием, потом рефакторинг на иммутабельный + множество ручных оптимизаций, затем рефакторинг на Mobx, который производительный по умолчанию.

Пакеты​

  • mobx-log - Логгер для Mobx с поддержкой Proxy-форматтеров

  • mobx-shallow-undo - Undo & redo для Mobx
  • mobx-store-inheritance - Поддержка наследования для makeAutoObservable в Mobx
